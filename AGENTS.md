# Deskripsi aplikasi
Aplikasi ini berfungsi sebagai pembuat CV yang memudahkan pengguna dalam menghasilkan dokumen CV secara cepat dan rapi. Pengguna dapat mengunggah data atau file CV lama, kemudian aplikasi akan melakukan konversi, memformat ulang, dan menghasilkan CV baru dengan tampilan yang lebih profesional. Aplikasi juga menyediakan fitur generate otomatis berdasarkan data yang diinput, sehingga pengguna dapat membuat CV dari awal tanpa harus memformat manual.

Jadi ini tuh aplikasi unik jadi conver ke aplikasi berdasarkan kode

# Tech stack
Base: Next js + Cloudflare + Tailwind + Shadcn + Unicorn Studio + il8n
Db: Supabase + Drizzzle
Auth: Supabase / Clerk
